```c
#include <stdio.h>
int main() {
    double a, b;
    scanf("%lf %lf", &a, &b);

    if (b == 0) {
        printf("0 으로 나눌 수 없습니다.");
        return 0;
    }
    printf("%f 를 %f 로 나눈 결과는 : %f \n", a, b, a / b);

    return 0;
}
```

# 논리 연산자
```c
#include <stdio.h>
int main() {
    int a;
    scanf("%d", &a);

    if(a >= 10 && a < 20) {
        printf("%d 는 10 이상, 20 미만인 수 입니다. \n", a);
    }
    return 0;
}
```
>배운 것.
>>**if, else if, else** 가 무엇 인지 알고 있습니다.

>>**논리 연산자** **&&, ||** 를 배웠습니다.

>>**!** 의 역할을 알고 있습니다.

>>**0 <= a <= 1** 을 **잘못된** 사용 예 입니다. 이 대신 **0 <= a && a <= 1** 과 같이 사용해야 합니다.
